# PrintGuard-Raspberry-Pi-Zero-2-W-Protective-Enclosure
I am building a compact, 3D printable enclosure for the Raspberry Pi Zero 2W featuring an OLED display window specifically 0.96" OLED I2C Display, USB C cutout, internal mounting posts, and passive cooling fins.
