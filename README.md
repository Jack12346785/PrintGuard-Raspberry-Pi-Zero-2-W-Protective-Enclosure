
# PrintGuard Raspberry Pi Zero 2 W Protective Enclosure

I am building a compact, 3D printable enclosure for the Raspberry Pi Zero 2 W

**Features:**

- 0.96" OLED I2C display window
- USB-C cutout
- Internal mounting posts
- Passive cooling
- an optional lid
